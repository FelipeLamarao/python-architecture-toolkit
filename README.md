# 🛠️ Python Architecture Toolkit

This repository contains essential architectural patterns and utilities built in Python, focusing on Clean Code and System Resilience.

## 📌 Features

* **Timer Decorator:** Monitors function execution time to identify performance bottlenecks.
* **Logger Decorator:** Automatically tracks inputs, outputs, and execution timestamps for auditing purposes.
* **Retry Decorator:** Provides resilience against network instability and API timeouts by automatically retrying failed operations.

## 🚀 Why this matters?
As an AI Solutions Integrator, separating business logic from infrastructure (like logging and monitoring) is crucial for building scalable and maintainable applications.
